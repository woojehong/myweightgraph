# 마웨그 쇼룸 V4 — 코드 네이티브 이펙트 사양서 (제안)

> 이미지는 GPT, **움직임·선·반응은 코드**. 이 문서는 이미지 없이 구현 가능한 범위만 다룬다.
> 아트바이블 「등급별 제작 밀도」를 코드 용어로 번역한 것이며, 등급 차이는 **컨셉이 아니라 밀도·정교함**이다.

---

## 1. 등급 = 밀도 (코드 기준 번역)

**모든 등급은 상시 루프한다.** 조건부·트리거·이벤트 발동 없음. 항상 돌아가고 있어야 한다.

| 등급 | 레이어 | 루프 | 동시 진행 요소 | 입자 예산 | 특징 |
|---|---:|---|---|---:|---|
| 고급 | 1~2 | 2~3초 | 1가지 | ≤ 8 | 단순하지만 끊김 없는 생명감 |
| 희귀 | 2~3 | 3~4초 | 2가지 (위상 차) | ≤ 20 | 두 요소가 맞물려 계속 돎 |
| 영웅 | 3~5 | 4~6초 | 3~4가지 (다중 속도) | ≤ 40 | 전경·중경·후경이 각자 다른 주기로 |
| 전설 | 5~7 | 6~9초 | 5가지 이상 (다중 속도·다중 재질) | ≤ 80 | 긴 주기 안에 기승전결이 계속 순환 |

**핵심 원칙**: 등급이 올라갈수록 *더 시끄러운* 게 아니라 **동시에 살아 움직이는 요소가 많고, 각 요소가 서로 다른 주기로 겹쳐 돌아 눈이 물리지 않는다.**

- 고급이 "한 가지가 깜빡인다"면, 전설은 "다섯 가지가 각기 다른 속도로 동시에 흐르고 그 합이 6~9초 주기로 기승전결을 이룬다."
- 긴 루프 안에 강약을 넣어 **상시 재생인데도 단조롭지 않게** 만든다. (예: 전설 8초 루프 = 잔잔 3초 → 고조 2초 → 절정 1초 → 잔상 2초, 그리고 다시 반복)
- 상시 재생이므로 **밝기·대비 상한 준수가 더 중요하다.** 계속 보이는 화면이라 조금만 세도 피로해진다.

### `ambient_effect`를 코드로 전환 제안

현재 계획은 webp 이미지지만, 코드가 명백히 우월하다:

- **안전영역 자동 준수** — `S5`(중앙 72%×68% 불투명 금지)를 차트 실제 픽셀 기준으로 계산해 회피. 이미지는 크기가 바뀌면 안전영역이 깨진다.
- 해상도 독립, 파일 용량 0, 데이터 뒤 z-order 보장
- **데이터 반응 가능** (아래 5장) — 이미지로는 절대 불가능

## 3. `line_style` 12종

체중 선 자체. Chart.js 그라디언트 스트로크 · segment API · 커스텀 플러그인으로 구현.
공통 규칙: **선의 실제 좌표는 절대 왜곡하지 않는다.** 장식은 선 주변에만.

### 고급
1. **잉크 실선** — 균일 두께 + 가장자리 미세 번짐. 아주 느린 농도 호흡.
2. **점묘 파선** — 파선 + 데이터 지점마다 미세 점. 점이 순차로 옅게 명멸.
3. **유리 실** — 본선 + 얇은 하이라이트 이중선. 하이라이트가 천천히 미끄러짐.

### 희귀
4. **흐르는 광맥** — 대시가 선 방향으로 흐름 + 은은한 글로우 맥동(위상 차).
5. **서리 맥** — 선 위 미세 결정 스파이크. 냉기 미광이 구간을 순환.
6. **쌍줄 나선** — 본선과 위상 다른 얇은 보조선이 교차하며 흐름.

### 영웅 (3~4요소 상시 병행)
7. **기억의 잔상** — 본선 + 지연 잔상선 2겹이 서로 다른 지연값으로 계속 따라붙음. 잔상 농도가 느리게 맥동.
8. **전류 회로** — 전류가 시작점→현재점을 **끊임없이** 주파. 굵은 전류 1개(느림) + 잔전류 2개(빠름)가 동시에 흐르고, 스파크가 랜덤 위치에 계속 튐.
9. **용광로 선** — 발열대가 선을 따라 계속 이동. 열원 뒤로 냉각 잔열이 남고, 표면에 미세 불티가 상시 피어오름.

### 전설 (5요소 이상 상시 병행)
10. **성운의 실** — 선을 이루는 입자가 계속 분해·재응집을 순환. 입자층·성운 안개·미광 점·궤도 파편이 각기 다른 속도로 동시 회전.
11. **시간의 직조** — 과거 구간이 끊임없이 되짚어 재생되며 현재까지 짜여 들어옴. 씨실·날실 2겹 + 직조 광점 + 완성 구간의 잔광이 동시 진행.
12. **창세의 궤적** — 빛이 현재점에서 태어나 전 구간을 역류하는 흐름이 계속 순환. 본선 발광 + 역류 광파 + 파생 입자 + 외곽 잔상 + 미세 굴절이 겹쳐 8초 주기로 강약을 이룸.

## 4. `ambient_effect` 12종 (전면 코드)

플롯 **바깥 테두리 band에만** 존재. 중앙 데이터 영역 침범 0. 최대 대비 12%.

### 고급
1. **미진** — 외곽에 부유 먼지 몇 알이 느리게 떠다님.
2. **잔광** — 모서리 4곳에 은은한 명멸.
3. **숨결 안개** — 하단 가장자리 옅은 안개가 호흡.

### 희귀
4. **반딧불 정원** — 모서리에서 반딧불 2~3마리가 궤적을 그리며 유영.
5. **심해 기포** — 좌우 가장자리를 따라 기포가 상승.
6. **낙엽 바람** — 상단 모서리를 잎이 사선으로 가로지름.

### 영웅 (3~4요소 상시 병행)
7. **빛 커튼** — 오로라 3겹이 각기 다른 속도로 상단을 계속 밀려왔다 물러남. 아래로 미세 광입자가 상시 낙하.
8. **유성우** — 유성이 끊이지 않고 모서리를 스침. 큰 것(느림) 1 + 잔유성(빠름) 3~4가 겹치고, 지나간 자리에 잔광이 남음.
9. **균열 미광** — 가장자리 균열 3곳이 서로 다른 주기로 열고 닫힘. 새어나온 빛이 주변에 번지고 미세 파편이 부유.

### 전설 (5요소 이상 상시 병행)
10. **성운 조류** — 외곽 성운층 2겹이 반대 방향으로 흐르며 색층을 계속 형성·소산. 별 점멸 + 미세 조류 입자 + 외곽 굴절이 겹침. (회전 금지)
11. **고대의 그림자** — 후면 거대 실루엣이 아주 느리게 순환 통과. **대비 8% 이하.** 전면에 안개층·부유 미진·모서리 미광이 각기 다른 주기로 동시 진행.
12. **창세의 파문** — 가장자리에서 파문이 계속 일어 둘레를 순회. 파문 3겹이 시차를 두고 겹치고, 교차 지점에 광점이 맺혔다 흩어짐.

## 5. 상시 재생인데 안 질리게 만드는 기법

상시 루프의 최대 적은 **단조로움**과 **눈의 피로**다. 조건부 발동 없이도 풍부하게 느껴지게 하는 장치:

### 5.1 서로소 주기 (핵심)
각 레이어의 주기를 **서로 나누어떨어지지 않게** 잡는다. 예: 3.1초 / 4.7초 / 7.3초.
→ 전체가 똑같이 반복되는 시점이 수백 초 뒤로 밀려서, 사용자는 **매번 조금씩 다른 조합**을 본다.
반대로 2초/4초/8초처럼 배수로 잡으면 8초마다 완전히 동일해져 즉시 지루해진다.

### 5.2 루프 내부 강약
등급이 높을수록 한 루프 안에 기승전결을 넣는다.
예) 전설 8초 = 잔잔 3초 → 고조 2초 → 절정 1초 → 잔상 2초 → 반복.
"항상 최대치"가 아니라 **항상 돌지만 세기가 오르내린다.**

### 5.3 준랜덤 변주
입자 생성 위치·크기·수명에 소량의 난수를 넣되 범위를 제한한다.
→ 매 루프 세부가 달라져 "녹화 영상 반복"처럼 보이지 않는다.

### 5.4 다중 속도 레이어
후경은 아주 느리게, 전경은 빠르게. 속도 차가 **깊이감**을 만든다.
등급별 동시 진행 요소 수(2장 표)가 곧 이 레이어 수다.

### 5.5 셀별 위상 분산
모아보기에서 여러 셀이 **동시에 같은 동작**을 하면 기계적으로 보인다.
셀마다 시작 위상을 랜덤 오프셋해서 각자 다른 순간을 재생하게 한다.

## 6. 안전·성능 규칙 (필수)

1. **데이터 우선** — 모든 이펙트는 데이터 선/마커보다 **뒤(z-order 아래)**. 예외는 지정된 포인트 이펙트뿐.
2. **안전영역** — ambient는 차트 실측 픽셀 기준 중앙 72%×68% 침범 0. 매 리사이즈마다 재계산.
3. **대비 상한** — 플롯 위 장식 대비 12% 이하, 전설 대형 실루엣은 8% 이하.
4. **모아보기 자동 강등** — 셀 4개 이상이면 **전 아이템을 고급 밀도로 다운시프트**. 6셀 × 전설 이펙트는 모바일에서 프레임이 무너진다.
5. **접근성** — `prefers-reduced-motion` 시 모든 모션 정지, 정적 표현으로 대체.
6. **탭 비활성 시 정지** — 백그라운드 탭에서 애니메이션 루프 중단(배터리).
7. **입자 예산 준수** — 2장 표 상한 초과 금지.
8. **위상 분산** — 여러 셀이 동시에 같은 동작을 하지 않게 셀마다 랜덤 시작 오프셋.
9. **상시 재생 전제의 밝기 억제** — 계속 보이는 화면이므로 순간 최대 밝기를 이벤트형보다 낮게 잡는다.

## 7. 테마 계열 (사용자 취향 기반)

**등급 = 퀄리티**이므로 테마는 등급과 무관하다. **같은 테마가 4등급 전부에 존재**하며, 등급이 오를수록 그 테마가 더 정교해질 뿐이다.
사용자·친구 그룹의 관심사: **워크래프트풍 하이 판타지 / 삼국지 / SF 전략 / 스포츠(야구·축구·테니스·레슬링)**

### 계열 목록

| 계열 | 시각 언어 | 이펙트 소재 |
|---|---|---|
| **A. 하이 판타지** | 룬, 마력, 원소, 고대 유물 | 룬 점등, 마력 흐름, 원소 파동, 소환진 |
| **B. 삼국지·동양 전략** | 죽간, 군기, 병법도, 인장 | 먹 번짐, 깃발 나부낌, 진형 이동, 낙관 인장 |
| **C. SF 전략** | 홀로그램, 에너지장, 군체, 정신체 | 스캔라인, 홀로 격자, 에너지 충전, 군체 유동 |
| **D. 스포츠** | 경기장 조명, 궤적, 스코어보드, 트로피 | 조명 점화, 공 궤적 잔상, 전광판 명멸, 함성 파동 |

### 등급 = 퀄리티 예시 (같은 테마, 4등급)

**「삼국지」 계열이 등급에 따라 어떻게 정교해지는가:**

| 등급 | 구현 |
|---|---|
| 고급 | 먹선 하나가 느리게 번졌다 옅어짐 (1레이어) |
| 희귀 | 먹 번짐 + 깃발 자락이 다른 주기로 나부낌 (2레이어) |
| 영웅 | 먹 번짐 + 깃발 + 배후 진형 광점이 이동 + 미세 낙관 점등 (4레이어, 다중 속도) |
| 전설 | 위 전부 + 병법도 격자가 짜였다 풀림 + 화공 잔열 + 인장 낙인 광파, 8초 주기 기승전결 (6레이어) |

→ **컨셉은 같고 밀도만 다르다.** 이것이 이 프로젝트의 등급 원칙이다.

### 저작권 주의

아트바이블 1장 원칙(특정 상용 게임 복제 금지)은 이펙트에도 적용한다.
- 삼국지·스포츠는 안전 (역사·일반 소재)
- 워크래프트풍·SF 전략은 **장르적 기대감만 공유**하고 특정 게임의 고유 문장·유닛·색 조합·명칭은 사용하지 않는다.
- 이펙트는 추상적 움직임이라 이미지보다 위험이 낮지만, **명칭에 고유명사를 넣지 않는다.**

## 8. 제안 진행 순서

1. **`line_style` 12종 구현** — 이미지 의존 0. 지금 즉시 착수 가능. GPT 이미지 작업과 완전 병렬.
2. **`ambient_effect` 12종 구현** — 코드 전환 승인 시 착수. 역시 이미지 의존 0.
3. **`emoji_border`** — 기존 80종에서 12종 선별·승격 (신규 제작 최소).
4. 이후 이미지 도착 순서대로 `motionLayer` 부착 (graph_skin → point_marker → companion → trophy → card_theme).

**1~3번은 GPT 이미지가 단 한 장도 없어도 완성 가능하다.** 이미지 대기 중 병목 없이 진행할 수 있는 구간.
