# 마웨그 쇼룸 V4 이미지 아트 바이블

## 1. 제작 목표

쇼룸 상품은 CSS 색상 프리셋이 아니라, 실제로 소유하고 과시할 가치가 있는 **고급 판타지 게임 아트 에셋**이다. 특정 상용 게임의 캐릭터·문장·장비를 복제하지 않고, 장르적 기대감만 공유한다.

- 기본값은 상품 96개와 별도로 제공한다. 기본값에는 등급·가격·과시 연출이 없다.
- 각 범주는 `고급/희귀/영웅/전설` 각 3개, 총 12개다. 전체 8범주 96개다.
- 등급 차이는 색이 아니라 제작 밀도, 재질 수, 분리 레이어 수, 동작의 사건성과 잔상으로 드러낸다.
- 이미지 안에는 그래프, 수치, 글자, UI 버튼을 그리지 않는다. 실제 데이터/UI가 이미지를 덮어쓴다.
- 모든 시안은 3종 비교 시트로 승인받은 뒤, 선택된 칸을 개별 고해상도 에셋으로 재생성한다. 시트를 잘라 제품에 쓰지 않는다.

## 2. 공통 미술·안전 규격

### 데이터 안전영역

- `S1 그래프 배경`: 메인 플롯의 실제 내부 픽셀 크기와 이미지 출력 크기를 1:1로 일치시킨 단일 16:9 배경이다. `object-fit:fill`로 왜곡하거나 `cover`로 자르거나 `contain`으로 회색 여백을 만들지 않는다. 생성 원본부터 정확히 16:9이며 중앙 가로 76% × 세로 70%에 불투명 장식, 강한 발광, 고대비 무늬를 두지 않는다.
- `S2 프로필 헤더 테두리`: 카드 배경이나 공간이 아니다. `이모티콘 + 이름 + 칭호` 프로필 헤더의 바깥 윤곽만 감싸는 투명 프레임이다. 내부는 완전 투명하고 텍스트·초상 콘텐츠 침범은 0px다.
- `S3 포인트`: 접점 중심 반경 18px 이내. 라벨 방향에는 12px 여백을 둔다.
- `S4 우측 상단`: 동반자는 우측 상단 14% × 22% 전용 둥지 안에만 존재한다.
- `S5 공간 후면`: 중앙 가로 72% × 세로 68%에는 불투명 픽셀 금지. 주변부 또는 데이터보다 뒤인 저대비 배경만 허용한다.
- `S6 좌측 상단`: 트로피는 좌측 상단 12% × 20% 전용 선반 안에만 존재한다.
- `S7 초상`: 얼굴 핵심부를 중앙 64% 안에 배치하고 프레임 없이 제작한다.
- `S8 초상 프레임`: 중앙 구멍 지름 68% 이상, 장식은 바깥쪽으로만 뻗는다.

### 등급별 제작 밀도

| 등급 | 실루엣/재질 | 레이어 | 동작 기준 |
|---|---|---:|---|
| 고급 | 한 가지 명확한 소재와 단순한 비대칭 포인트 | 1~2 | 호흡·점멸 등 1동작, 2~3초 |
| 희귀 | 두 소재 결합, 보조 문양 또는 작은 생명성 | 2~3 | 순환 2동작, 3~4초 |
| 영웅 | 서사적 물체 조합, 전경·중경·후경 분리 | 3~5 | 시작-절정-복귀가 있는 4~6초 사건 |
| 전설 | 상징적 대형 실루엣, 다중 재질과 공간 깊이 | 5~7 | 등장-변형-절정-잔상, 6~9초. 과도한 상시 운동 금지 |

## 4. 3종 이미지 시트 생성 프롬프트

### 공통 프롬프트 머리말

아래 각 행의 `A/B/C`를 이 공통 문장 뒤에 붙여 사용한다.

> Use case: stylized-concept. 마웨그 쇼룸용 오리지널 고급 판타지 RPG 게임 아트, 가로형 3패널 비교 시트, 세 패널 동일 크기와 동일 카메라, 각 패널에 독립된 한 상품. 세 상품은 색상 변형이 아니라 실루엣·재질·서사가 확연히 달라야 한다. 풍부한 손그림 질감, 정교하고 절제된 광원, 현대적인 게임 UI 에셋 품질. 특정 유명 게임의 캐릭터·문장·장비를 복제하지 말 것. 글자, 숫자, 그래프, 로고, 워터마크 없음.

카테고리별 후미 제약:

- GS: `16:9 정면 기록 매체. 중앙 76%×70%는 어둡고 무문양인 데이터 안전영역.`
- CT: `프로필 헤더용 가로형 투명 테두리. 원형 초상 슬롯과 이름·칭호의 두 줄 텍스트 안전영역을 바깥에서 연결하되 내부 콘텐츠 침범 0px. 배경·방·풍경·판넬 채움 없음.`
- PM: `정면 소형 마커. 균일한 #FF00FF 배경, 그림자 없음, 중앙 접점 하나.`
- CP: `우측 상단용 작은 3/4 전신 마스코트. 균일한 #FF00FF 배경, 바닥 그림자 없음.`
- AE: `16:9 주변부 효과 키프레임. 중앙 72%×68%는 완전히 비우고 낮은 대비.`
- TR: `좌측 상단 선반용 단일 트로피. 균일한 #FF00FF 배경, 바닥 그림자 없음.`
- PE: `어깨 위 3/4 감정 초상, 프레임 없음, 48px에서 얼굴이 읽힘.`
- EB: `원형 초상 프레임, 중앙 구멍 68% 이상, 균일한 #FF00FF 배경, 얼굴 없음.`

| EB 희귀 | 항해매듭 조타환 | 양봉가의 황금망 | 모자이크 공작환 | EB |
| EB 영웅 | 왕벌의 육각궁 | 극단장의 가면막 | 유성우 리본궁 | EB |
| EB 전설 | 만조의 조개궁 | 종이궁전의 접이문 | 영원의 자수성곽 | EB |

프롬프트의 상품명만으로 생성하지 말고, 반드시 3장의 해당 아트 매트릭스 행에서 `실루엣·재질·동작`을 A/B/C 설명에 함께 복사한다. 움직임이 있는 상품의 시트는 정지 대표 포즈를 뽑고, 승인 후 본체·가동부·발광부를 개별 생성한다.

## 5. 에셋 분리 및 납품 규격

| 범주 | 정적 본체 | 동작 레이어 | 권장 제품 포맷 |
|---|---|---|---|
| 그래프 스킨 | 1536×864 | 동일 캔버스 1~4장 | WebP + alpha WebP/PNG |
| 카드 테마 | 프로필 헤더 실측 비율의 투명 프레임 | 외곽 가동부·광원 1~4장 | PNG + alpha WebP |
| 포인트 마커 | 256×256 | 발광·가동부 1~3장 | PNG/WebP |
| 동반자 | 512×512 | 몸/눈/가동부 2~5장 | PNG 시퀀스 또는 WebP |
| 공간 효과 | 빈 1536×864 캔버스 | 후면/주변부 2~6장 | alpha WebP |
| 트로피 | 512×512 | 발광·가동부 1~4장 | PNG/WebP |
| 프로필 이모티콘 | 512×512 | 눈·입·광원 1~3장 | WebP |
| 이모티콘 테두리 | 768×768 | 가동부·광원 1~5장 | PNG/WebP |

- 크로마키 피사체는 캔버스 가장자리와 최소 8% 간격을 둔다.
- 반투명 연기·유리·불꽃은 본체와 한 장으로 합치지 않는다. 별도 발광 레이어로 생성한다.
- 애니메이션은 무한히 산만하게 움직이지 않는다. 6~12초 휴지 후 한 번 재생이 기본이다.
- `prefers-reduced-motion`에서는 대표 정지 프레임만 표시한다.

## 6. 미술 QA 불합격 기준

1. 이름을 가렸을 때 같은 등급의 세 상품 실루엣을 구분할 수 없다.
2. 등급 상승이 채도·발광량 증가로만 표현된다.
3. 안전영역에 불투명 픽셀, 고대비 문양, 강한 섬광이 들어온다.
4. 프로필 프레임이 얼굴 또는 이모티콘 외곽을 가린다.
5. 48px 축소 시 포인트·초상·프레임의 핵심 형상이 무너진다.
6. 생성 이미지에 가짜 글자, 숫자, 그래프, UI가 포함된다.
7. 특정 유명 IP의 고유 캐릭터, 문장, 무기 실루엣과 혼동된다.
8. 전설 효과가 그래프보다 먼저 보이거나 데이터 판독 시간을 늘린다.
9. 크로마키 제거 후 색 테두리, 반투명 구멍, 잘린 장식이 남는다.
10. 시트 이미지를 단순 크롭해 최종 제품 에셋으로 사용한다.
