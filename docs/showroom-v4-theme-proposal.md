# 쇼룸 V4 테마 제안서 (컨펌용)

> **제작 전 승인 문서.** 승인 후 GPT 이미지 브리프 / 코드 구현으로 나눠 진행한다.
> 원칙: **정확한 고유명사는 쓰지 않되, 보면 바로 알아보게 연상시킨다.** (오마주·패러디 톤)
> 등급은 **컨셉이 아니라 퀄리티·밀도** 차이다. 같은 소재도 등급이 오르면 레이어가 늘어난다.

## 8. 컨펌 요청 사항

1. **6계열(A~F) 구성**이 마음에 드시는지 — 빼거나 추가할 계열
2. **계열 배분** — 지금은 등급마다 계열을 섞었다. 특정 계열을 더 늘릴지
3. **명칭 톤** — 지금 수준(연상되지만 고유명사 아님)이 적절한지
4. 승인 후 진행 순서: `ambient_effect` → `emoji_border` → 칭호 (내 작업) / GPT 브리프 별도 산출
