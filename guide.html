👟 걸음 수 자동 연동

아이폰 단축어(무료 기본 앱)로 매일 밤 그날의 걸음 수를 자동으로 마웨그에 보냅니다.
처음 한 번만 5분 정도 설정하면, 그 뒤로는 손댈 필요 없어요.

1️⃣ 내 연동 코드 만들기
불러오는 중...
2️⃣ 단축어 만들기 (아이폰)
  1. 단축어 앱 열기 → 오른쪽 위 ＋ 눌러 새 단축어 만들기. 이름은 "걸음 보내기" 정도로.
  2. 검색창에 "건강"을 입력하고 건강 샘플 찾기 동작 추가.
    동작 안 파란 글씨를 눌러 이렇게 설정:
    · 유형: 걸음 (걸음 수)
    · "모든 건강 샘플" 부분을 눌러 필터 추가 → 시작 날짜 → 오늘
    · 정렬 기준: 없음, 제한: 끔 (하루 걸음이 여러 조각으로 나뉘어 저장되기 때문에 전부 가져와야 해요)
  3. 걸음 조각들을 하나로 합치기: 통계 계산 동작 추가 (검색창에 "통계" — 안 보이면 "계산"으로 검색) → 연산: 합계, 입력은 자동으로 위의 건강 샘플이 잡힙니다.
  4. URL 동작 추가 → 위 1번 카드의 내 개인 URL을 붙여넣고, URL 맨 끝(steps= 바로 뒤)에 커서를 두고 키보드 위 변수 목록에서 통계 계산 결과 변수를 선택해 붙입니다.
    완성 예: ...&steps=통계 계산
  5. URL 내용 가져오기 동작 추가 → 방법: GET (기본값 그대로).
    ▶︎ 버튼으로 한 번 실행해보세요. 건강 데이터 접근 허용 팝업이 뜨면 꼭 허용! {"ok":true...}가 나오면 성공입니다.
⚠️ 자주 하는 실수
· 건강 접근 권한 팝업에서 "허용 안 함"을 누르면 걸음 수가 0으로 나와요. 설정 → 개인정보 보호 및 보안 → 건강 → 단축어에서 다시 켤 수 있어요.
· "건강 샘플 찾기"에서 필터(시작 날짜: 오늘)를 안 걸면 옛날 걸음까지 전부 합산됩니다.
3️⃣ 매일 밤 자동 실행 걸기
  1. 단축어 앱 하단 자동화 탭 → ＋ (또는 "새 자동화") → 개인용 자동화.
  2. 시간 선택 → 매일, 시각은 오후 11:45 권장 (자정 넘기면 다음 날 걸음으로 저장돼요).
  3. 실행 옵션에서 즉시 실행 선택 (구버전 iOS에선 "실행 전에 묻기" 끄기) → 알림도 꺼도 됩니다.
  4. 실행할 항목으로 방금 만든 "걸음 보내기" 단축어 선택 → 완료.
  5. 다음 날 아침, 그래프와 업적 페이지에서 걸음 수가 자동으로 들어왔는지 확인해보세요. ✅
⚠️ 알아두기
· 실행될 때 화면 상단에 자동화 알림이 잠깐 뜨는 건 정상이에요.
· 밤 11:45에 폰이 완전히 꺼져 있으면 그날은 건너뜁니다 — 다음 날 아침에 단축어를 한 번 수동 실행하면 오늘 날짜로 들어가니, 놓친 날은 관리자에게 말해주세요.

🤖 안드로이드 친구들: 아직 자동 연동은 아이폰만 지원해요. 당분간은 수동 입력을 부탁드리고, 삼성헬스 CSV 가져오기는 추후 지원 예정입니다.

4️⃣ 바로 테스트해보기

아래 버튼을 누르면 내 개인 URL로 걸음 수 1234를 실제 전송합니다. 오늘 날짜 기록에 저장되니, 진짜 걸음이 이미 기록된 날엔 자동화가 밤에 다시 덮어써 줄 거예요.

🔒 어떤 데이터가 전송되나요?

전송되는 건 그날의 걸음 수 숫자 하나뿐입니다.
심박수, 위치, 수면 등 다른 건강 데이터에는 아예 접근하지 않고, 서버에도 걸음 수와 날짜만 저장돼요. 연동 코드는 내 계정에만 걸음 수를 기록할 수 있게 해주는 열쇠이니 다른 사람에게 공유하지 마세요.
